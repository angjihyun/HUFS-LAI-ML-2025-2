# 리뷰 감정 분석 기반 개인 맞춤 화장품 추천 시스템

## 1. 해결하고자 하는 문제
화장품을 구매할 때 단순한 별점이나 평균 평점만으로는 나에게 맞는 제품을 찾기 어렵다.
같은 제품이라도 “향이 진해서 좋다”와 “향이 진해서 싫다”처럼 서로 다른 평가가 섞여 있어,
나와 비슷한 감정 성향을 가진 사람들이 좋아한 제품을 파악하기 힘들다.
나 역시 “무겁지 않고 산뜻한 제형”을 선호하지만 리뷰를 일일이 읽어도 내 취향에 맞는 제품을 찾기 어려웠다.
따라서 내가 직접 작성한 리뷰를 입력하면,
나와 감정적으로 비슷한 사용자들이 긍정적으로 평가한 제품을 추천해주는 시스템을 만들고자 한다.
실제로 화장품을 구매하기 전마다 내 리뷰를 입력하고 결과를 확인할 계획이며,
세럼, 쿠션, 선크림, 토너, 립 등 카테고리별로 최소 다섯 번 이상 사용해볼 예정이다.

## 2. 제안하는 솔루션
사용자가 작성한 리뷰 문장의 감정을 분석하고
감정적으로 유사한 리뷰를 찾아 그 리뷰가 달린 제품을 추천하는 간단한 웹 서비스이다.
입력한 리뷰가 “무겁다, 답답하다”처럼 부정적인 내용이면
데이터셋 내에서 “가볍다, 산뜻하다”고 긍정 평가된 제품을 찾아 추천한다.
결과는 Gradio 대시보드에서 감정 분석 그래프와 함께 보여준다.
주요 기능은 감정 분류, 유사 감정 리뷰 탐색, 추천 제품 리스트 출력이며
사용자는 웹 화면에서 직접 리뷰를 입력하고 결과를 확인하는 방식으로 사용한다.

## 3. ML 활용 계획
공개된 Naver Shopping Review Dataset(한국어 리뷰 약 20만 개)을 사용한다.
리뷰 텍스트를 정제하고 별점으로부터 감정 레이블(긍정, 중립, 부정)을 만든다.
감정 분류는 TF-IDF + Logistic Regression으로 기본 모델을 학습하고
성능 향상을 위해 KoBERT를 파인튜닝한다.
이후 Sentence-BERT로 리뷰 문장을 임베딩하고
입력 리뷰와 데이터셋 리뷰 간 cosine similarity를 계산해
가장 유사한 긍정 리뷰가 달린 제품을 추천한다.
감정 분류는 Macro-F1 score로 추천 정확도는 Recall으로 평가하며
실제 사용을 통해 추천 결과가 선호도와 얼마나 일치하는지도 함께 검증할 예정이다.
